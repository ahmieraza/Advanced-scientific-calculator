# Advanced Scientific Calculator

A powerful HiPER-style scientific calculator built with React, TypeScript, and Tailwind CSS.

## Features

- **Comprehensive Mathematical Functions**
  - Trigonometry: sin, cos, tan (with inverse functions)
  - Logarithms: log, ln, with inverse (10^x, e^x)
  - Advanced operations: factorial, nPr, nCr
  - Power functions: x², xʸ, √x, ⁿ√x
  - Calculus tools: ∫dx, d/dx

- **Multiple Modes**
  - Degree/Radian mode toggle
  - SHIFT mode for secondary functions
  - ALPHA mode for variables
  - Hyperbolic mode

- **Smart Features**
  - Auto-complete parentheses
  - Memory storage (M+, RCL)
  - Last answer recall (Ans)
  - Scientific notation (Exp)
  - Real-time calculation

- **User Experience**
  - Mobile-friendly responsive design
  - Compact layout fits any screen
  - Touch-optimized buttons
  - Visual mode indicators

## Installation

```bash
npm install
npm run dev
```

## Usage

1. Use number buttons for input
2. Select mathematical functions
3. Toggle SHIFT/ALPHA for secondary functions
4. Press = to calculate
5. Use Ans to recall last result
